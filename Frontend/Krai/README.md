# KRAI - Responsive Adventure Website

This is a responsive landing page for an adventure-themed website named "KRAI". The project is built with HTML and SCSS/CSS, and it is fully responsive.

## Features

- **Responsive Design:** The website is designed to work on various screen sizes, with a specific breakpoint for devices wider than 800px.
- **Interactive UI:** Hover effects on activity cards and buttons to enhance user experience.
- **Single-Page Layout:** All content is presented on a single, scrollable page.
- **Custom Fonts and Icons:** The project uses custom fonts and icons from Remixicon to create a unique visual identity.

## Technologies Used

- **HTML5:** For the structure of the website.
- **SCSS/CSS3:** For styling, with features like variables, mixins, and nested rules.
- **Remixicon:** For icons used throughout the website.

## How to Use

To view the website, simply open the `index.html` file in your web browser.

## Project Structure

- `index.html`: The main HTML file.
- `style.scss`: The main SCSS file containing all the styles.
- `style.css`: The compiled CSS file.
- `Archa-Manic-Demo-Thin.ttf`, `Japan-Daisuki.otf`, `Sublem.ttf`: Custom fonts.
- `camp.png`, `hike.png`, `man.png`, `sight.png`: Image assets.
